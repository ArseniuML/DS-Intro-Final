Для запуска проекта:

Рядом с папкой Final_work создать папку Final_work_data и положить туда файлы ga_hits.csv и ga_sessions.csv

Запустить скрипт Final_work/source/create_ml_data.ipynb. Он создаст файл Final_work_data/ml_data.csv

Разведочный анализ в файле eda.ipynb

Тренировка и сравнение моделей - model.ipynb. У меня скрипт полностью работает около 2 часов. В конце он создаст файл Final_work/data/model.pkl (сама финальная модель создается несколько минут)

Localhost Web App в файле source/predict.py

Презентация - presentation/ds_final_work.pptx
